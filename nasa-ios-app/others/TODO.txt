APOD - https://apod.nasa.gov/apod/astropix.html
Details: {
    Pode ser uma imagem ou um vídeo (verificar se é preciso ou nao a thumb), no filtro por range verificar se existe limite de range
    Descriçao com scrollable view
}
Functionalities: {
	Ir buscar do dia (com uma galeria de poder ver os anteriores)
	Filtrar por: data, range (data inicio e/ou data fim), random date (count = 1)
}

Earth - https://api.nasa.gov/planetary/earth/imagery?lon=-95.33&lat=29.78&date=2018-01-01&dim=0.15&api_key=DEMO_KEY
Details: {
    coordenadas e data obrigatorio, dim default = 0.15
}
Functionalities: {
    fazer um mapa onde o user escolhe o local e aparece no ecra o resultado, e sendo possível escolher o dia, senao pega hoje
    a pessoa introduzir as coordenadas e a data
    fazer zoom (alterar dim)
}

EONET - https://worldview.earthdata.nasa.gov > https://eonet.gsfc.nasa.gov/docs/v2.1 > https://eonet.gsfc.nasa.gov/api/v2.1/events/EONET_6353
Details: {
    Ao pressionar o icon aparecer o nome da categoria
}
Functionalities: {
    Feed de eventos (mais recente primeiro, 20 primeiros)
    Filtrar evento: {
        por categorias (ir buscar o endpoint das categorias para carregar)
        por paises?
        ativos ou nao ativo
        tracking no caso de eventos que se movam (que tenham mais de 1 coordenada)
    }
}
